Задание 3

Задание: Напишите обобщенный класс Calculator<T>, который имеет статический метод T Add(T x, T y), который возвращает сумму двух значений типа T. Затем напишите ограничение для этого класса, чтобы он мог работать только с типами, которые имеют parameterless конструктор (new()). Затем напишите метод T Zero(), который возвращает нулевое значение типа T с помощью parameterless конструктора.
